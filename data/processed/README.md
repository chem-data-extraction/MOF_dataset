# Processed data

This folder contains the final publication tables generated by the local pipeline.

## Files

- `adsorption_measurements.csv`: normalized measurement table
- `mof_materials.csv`: normalized MOF material table
- `dataset.csv`: flat compatibility export aligned with `specs/dataset_schema.json`

The normalized tables are the main publication format. `dataset.csv` is retained so automated validators and quick preview tools can read one flat file.

## Regeneration

Run:

```bash
py -3 scripts/build_dataset.py
py -3 scripts/clean_dataset.py
py -3 scripts/validate_project.py
```

The processed files should be regenerated from scripts after parser or schema changes.
